# Jimmy Choo 官网前端设计风格解读

为了将当前的数据分析系统（基于 Tailwind CSS 构建）升级为符合 Jimmy Choo 品牌调性的“奢华、极简、专业”视觉体验，我们对 [Jimmy Choo 官方网站](https://www.jimmychoo.com/hk/en_HK/home) 的前端设计语言进行了拆解，提炼出以下核心设计规范（Design Tokens）：

## 1. 色彩规范 (Color Palette)
奢侈品电商的色彩极度克制，通过极简的黑白灰来凸显商品图片和核心数据，杜绝高饱和度彩色。

*   **Primary (主色)**:
    *   `#000000` (纯黑) - 用于主标题、主要正文、核心按钮背景。
    *   `#FFFFFF` (纯白) - 用于页面背景、卡片背景，大面积留白。
*   **Secondary (辅助/中性色)**:
    *   `#757575` (深灰, Tailwind `gray-500`) - 用于次要文本、说明性文字（如款式数量、辅助说明）。
    *   `#E5E5E5` (浅灰, Tailwind `gray-200`) - 用于极细的分割线（Divider）、表格边框。
    *   `#F9F9F9` (极浅灰, Tailwind `gray-50`) - 用于斑马线表格背景、占位图底色、Hover 状态。
*   **Accent (强调色 - 替换当前的 Amber)**:
    *   Jimmy Choo 官网不使用类似当前 `amber-400` 这样的彩色高亮。
    *   **升级建议**：将原本用作“本品强调”的琥珀色，替换为**黑白反白（黑底白字）** 或 **加粗的黑色边框**，以符合高冷调性。

## 2. 排版与字体 (Typography)
字体的选择和排版是奢侈品牌官网的灵魂，注重几何感和呼吸感。

*   **Font Family (字体族)**:
    *   官网主要使用定制的几何无衬线字体（Geometric Sans-Serif），类似 `Futura`, `Montserrat` 或 `Proxima Nova`。
    *   在我们的项目中，可以使用 Tailwind 的 `font-sans` 并配合大写字母（Uppercase）与字间距（Tracking）。
*   **Headings (标题)**:
    *   **全大写 (UPPERCASE)**：导航、Tab栏、列名通常使用全大写。
    *   **字间距 (Letter Spacing)**：标题普遍增加字间距（如 `tracking-widest` 或 `tracking-wider`），增加高级感。
    *   **字重 (Weight)**：倾向于使用 `font-light` (300) 或 `font-medium` (500)，极少使用特别粗的黑体。
*   **Body (正文与数据)**:
    *   小巧精致。数据字号可略小（如 `text-sm` 或 `text-xs`），但必须保证清晰。

## 3. 组件样式 (Components)
摒弃现代 Web 2.0 中常见的圆角、厚重阴影和彩色背景，回归纸质印刷排版（Editorial Design）的质感。

*   **Buttons & Tags (按钮与标签)**:
    *   **直角 (Sharp Corners)**：废弃大圆角（`rounded-lg`, `rounded-xl`），改为直角（`rounded-none`）或极小的微圆角（`rounded-sm`）。
    *   **Ghost 风格**：次要按钮和本品标签，使用白底黑框（`border border-black text-black bg-transparent`）。
    *   **Solid 风格**：主操作或当前选中的 Tab，使用黑底白字（`bg-black text-white`）。
*   **Cards & Modals (卡片与弹窗)**:
    *   **去边框/去阴影**：商品卡片（ProductCard）去除外围边框（`border`）和阴影（`shadow-md`）。卡片之间的区分纯靠**留白（Whitespace）**。
    *   **大图展示**：图片比例多为纵向（如 `aspect-[3/4]`），图片贴边，下方文字左对齐，去除彩色 Tag。
    *   弹窗（Modal）的背景遮罩要深（如 `bg-black/60`），弹窗本身纯白，直角边缘。
*   **Tables (数据表格)**:
    *   极简的水平分割线（仅保留 `border-b border-gray-200`），**去掉垂直分割线**。
    *   表头（th）全大写，灰色小字，左对齐（除了数字右对齐/居中）。
    *   行高要大（如 `py-4`），让数据有呼吸感。

## 4. 空间与布局 (Spacing & Layout)
*   **Whitespace (留白)**：是提升高级感的最有效手段。增加各个区块之间的 Margin（如 `mb-12`, `py-8`）。
*   **对齐 (Alignment)**：严格的左对齐或中轴对称，避免视觉焦点跳跃。

## 🚀 针对当前系统 (`CategoryAnalysis.tsx`) 的升级行动指南

若要将目前的 UI 升级至上述风格，建议对代码进行以下改造：

1. **色彩去色化 (De-colorization)**：
   * 将所有 `amber-50`, `amber-300`, `amber-400`, `amber-600` 等彩色替换为 `black`, `gray-900`, `gray-200`。
   * “本品”的标识牌，从 `<span className="bg-amber-400 text-white rounded">` 改为 `<span className="border border-black text-black bg-white uppercase text-[10px] tracking-widest px-2 py-0.5">`。
2. **去除圆角与阴影 (Flat & Sharp)**：
   * 将 `rounded-lg`, `rounded-xl`, `rounded-2xl` 全部降级为 `rounded-none` 或极小的 `rounded-sm`。
   * 删除卡片容器的 `border` 和 `hover:shadow-md`。
3. **Tab 样式重构**：
   * 选中的 Tab：底部的黑色下划线，或者黑底白字的直角块。
   * 未选中的 Tab：浅灰色，Hover 时变黑。全大写，加宽字间距。
4. **表格 (Table) 精简**：
   * 表头：`uppercase text-xs tracking-wider text-gray-500 font-normal`。
   * 行背景：取消默认的斑马线，鼠标 Hover 时使用极浅灰（`hover:bg-gray-50`）高亮整行即可。
5. **图片比例调整**：
   * 鞋履图片的展示在奢侈品网站通常不是完美的正方形 (`aspect-square`)，而是偏修长的矩形（如 4:5 或 3:4）。建议调整 `ProductCard` 的图片容器比例。

以上解读可作为接下来进行代码重构与 UI 升级的设计蓝图。
